# ai-llm-hub
AI 大模型资源中心 | 国内外大模型导航、能力对比、开发知识库

## Slogan
- 聚合全球大模型，一站式查询与选型
- 懂模型，更懂你需要哪一个
- 开发者必备的 AI 大模型导航与知识库

## 项目背景
日常开发中经常需要使用各类 AI 大模型进行需求分析、代码生成、调试、Bug 修复等工作，但不同模型擅长领域不同，查找信息分散、对比成本高、使用不便。为此，搭建一个集中、全面、易用的 AI 大模型资源平台，方便开发者快速查询、了解、对比和选型。

## 项目定位
- 面向开发者、产品、AI 爱好者的**大模型资源聚合平台**
- 提供模型介绍、能力特点、适用场景、使用指南、工具推荐
- 一站式解决“不知道用哪个模型、去哪里查、怎么用”的问题
